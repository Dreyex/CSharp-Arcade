Projekt erstellen

Initialisierung des Projekts

Um ein C#-Projekt zu erstellen muss im Zielordner die Konsole geöffnet werden. (CMD/Powershell/etc.)
In die Konsole müssen dann folgende Befehle eingegeben werden:

dotnet new console

Es wird ein neues Projekt erstellt. Nach der Standardvorlage für eine Konsolenanwendung.
Es gibt weitere Standardvorlagen die auf der Website von Microsoft zu finden sind, sowie weitere Argumente für den Befehl.



Projekt ausführen

Um das Projekt nun auszuführen wird folgender Befehl verwendet:

dotnet run

Das Projekt wird nun ausgeführt.
In der Standardvorlage wird die Ausgabe "Hello World!" in der Konsole ausgegeben.
Um den Code zu bearbeiten empfehle ich die Verwendung eines Texteditors wie Visual Studio Code .
Dieser Editor hat einige Vorteile. Mit der Extension "Code Runner" kann der Code direkt in der Konsole ausgeführt werden über die Standard Schaltfläche "Run Code".
Das ist um einiges bequemer als jedes mal die Konsoleneingabe einzutippen.
Ebenso gibt es die Möglichkeit eine von Microsoft selbst erstellte Extension für C# zu installieren.
Diese bietet einige Vorteile wie z.B. die automatische Erkennung von Fehlern und die automatische Erstellung von Klassen und Methoden.
Um diese zu finden einfach in der Suche bei den Extensions "C#" eingeben.
